ソーシャル断ち、20日目。

G+ には Picasa / Blogger からシェアするという技で落ち着きつつある。赤い四角問題は引き続き。

昼のひとりごと：querySelectorAll、かー。
formに入力した要素を取得するのに、onclickでfunction実行したいだけなのにbutton要素つかっていきなりはまった。何もないとこにsubmitしてリロードしてたせいで値が消えてたみたい。

夜はGASコードラボに初参加。ぎりぎりで繰り上がり、ありがたし。a2cさんに再会、そしてふたりいるうさぎの人にお会いできたので区別できるようになった！GASはgeeklatte.com計画でちょっといじったので、そしてちょっとでもいじったことあったら「がんがん組」ってことでがんがん組に入ったけど、理解がついてくのがむずかしかった。最後の30分くらいで、じゃあ好きなもの作ってーのコーナーで、うーん、何ができて何がしたいんだろ、ってもやもやしてたら時間切れ。
もやもやふわふわしてる。
  • 基本操作：File > Manage Versions で Version つくってから（Description には commit log 的なのを）Publish > Deploy as web app 
  • Test web app for your latest code. で Publish せずに最新の状態が確認できるよ
  • doGet() が肝心
  • Ctrl + Space で auto complete しようぜ
  • HtmlTemplate たのしそう！だけど getCode() と getContent() のとこ聞く余裕なかった（その前のやつに苦戦してて）
  • google.script.* よくわかんなかったのでこれ読む https://developers.google.com/apps-script/html_service#GoogleScriptAPI
  • サーバ側とクライアント側がふわふわしてる
